F1 2019 game add-on for Splunk
==============================

Add-on built to collect F1 2019 game data via UDP packets then convert into JSON before sending to Splunk.

Leveraging code from -

1) https://pypi.org/project/f1-2019-telemetry/ to capture the data into python classes
2) https://github.com/rinatz/ctypes_json to convert python classes into JSON

Add-on logo from https://www.svgrepo.com/svg/147773/formula-1-car-facing-right

This is an add-on powered by the Splunk Add-on Builder.
